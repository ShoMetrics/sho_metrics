## Create Your First Key

1. Open the Stream Deck app.
2. Find the Sho Metrics actions.
3. Drag a metric action onto an empty key.
4. Choose the metric in the Property Inspector.
5. Watch the key update.

## What To Try First

Start with CPU usage or memory usage. These are easy to confirm from Task Manager while testing.
