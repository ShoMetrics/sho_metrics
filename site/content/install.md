## Install The Plugin

1. Download the Sho Metrics plugin package.
2. Open the package to install it into Stream Deck.
3. Restart Stream Deck if the actions do not appear.

## Install The Helper

1. Download the Sho Metrics helper.
2. Run the installer.
3. Keep the helper running while using hardware-backed metrics.

## Add A Metric

1. Open the Stream Deck app.
2. Drag a Sho Metrics action onto a key.
3. Select the metric and display style in the Property Inspector.
