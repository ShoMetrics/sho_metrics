Is ShoMetrics free?

Yes. ShoMetrics is free and open source.

Do I need a Stream Deck to run it?

As of today, yes — ShoMetrics runs inside the Stream Deck. You don't need the physical hardware, though: Elgato's Virtual Stream Deck works too, and it's free if you already own a Stream Deck device.

Which metrics are available?

CPU, GPU, memory, disk, and network out of the box, with more sources on the way. GPU metrics work natively on NVIDIA cards and on macOS; Intel and AMD GPUs need the helper installed for deeper readings.

Which systems does it support?

Windows 10, version 1809 or later (Windows 11 is recommended), and macOS 12 or later on Apple Silicon. ShoMetrics may run on other systems, but those are neither tested nor recommended.

What is the helper for?

The helper is a small Windows-native companion app that unlocks deeper hardware sensors, such as fan speeds, voltages, and per-component temperatures. It builds on the well-known LibreHardwareMonitor project. See the helper guide for setup.

Is it safe to monitor my system?

It's designed to be, with open source at its core, so you don't have to take that on trust. Being able to see all the way through a system monitor is a big part of why ShoMetrics exists: the code is public, and every release is built and published through GitHub Actions with no manual steps. If you have extra concerns, you can compare a download's hash against the release or build ShoMetrics yourself from source. Please note, the optional Windows helper installs a system driver to reach deeper sensors (see the helper guide). No software can claim to be completely free of bugs, but the project stays transparent so you can judge it on the code, not on a promise.

Will it slow my system down?

We build ShoMetrics on the principle that a monitor should stay out of your way, and put real effort into keeping overhead low. Monitoring always costs a little, but the aim is to keep it small enough that you don't notice it.
