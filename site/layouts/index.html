Which metrics are available?

The plugin currently exposes CPU, GPU, Memory, Disk, and Network actions.

Which systems does the plugin target?

The Stream Deck manifest targets Windows 10 or later and macOS 12 or later.

What is the helper for?

The Windows helper is used when richer hardware data is needed. Download and setup details live on the download page.
