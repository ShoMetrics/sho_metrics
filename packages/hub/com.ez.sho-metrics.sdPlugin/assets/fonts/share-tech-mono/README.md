# Share Tech Mono

Vendored Share Tech Mono font files for ShoMetrics old CRT theme rasterization.

- Source: https://github.com/google/fonts/tree/main/ofl/sharetechmono
- Files kept:
  - `ShareTechMono-Regular.ttf`
  - `LICENSE.txt`
- License: SIL Open Font License 1.1

Share Tech Mono is used for old CRT labels where a readable mono face keeps the
small Stream Deck text from becoming too noisy.
