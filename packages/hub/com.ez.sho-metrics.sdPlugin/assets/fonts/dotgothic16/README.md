# DotGothic16

Vendored DotGothic16 font files for ShoMetrics pixel-style theme rasterization.

- Source: https://github.com/fontworks-fonts/DotGothic16
- Downloaded artifact: `DotGothic16-Version1.101.zip`
- Files kept:
  - `DotGothic16-Regular.ttf`
  - `LICENSE.txt`
- License: SIL Open Font License 1.1

DotGothic16 is used as the primary bundled pixel UI font candidate for small
Stream Deck widget text.
