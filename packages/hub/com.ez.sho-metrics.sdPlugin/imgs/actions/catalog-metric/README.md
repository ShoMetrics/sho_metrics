# Advanced Sensor Icon

`icon.svg` is a hand-maintained Stream Deck action-list icon for the Advanced
Sensor action. It is intentionally not generated from the ShoMetrics brand
source because this action needs a generic hardware symbol, not the app mark.

The icon is based on Material Symbols Developer Board and remains white on
transparent ground for Stream Deck action-list contrast.
