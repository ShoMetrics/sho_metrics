2. [Stacked Metric Widget Plan](02-stacked-metric-widget-plan.md)

## Scope

This folder owns widgets that display more than one metric selection inside one
Stream Deck action.

Read this folder before changing multi-slot settings, multi-metric action
subscriptions, dense progress-list rendering, stacked metric rotation, slot
indicator rendering, or Property Inspector metric-slot editing.

HTTP custom metrics do not belong in this folder. They are source/target input
features: they define how a metric enters the system. Dense and Stacked Metric
define how existing metric selections are arranged inside one widget.
