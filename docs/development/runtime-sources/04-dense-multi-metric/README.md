This folder owns the first simultaneous multi-metric widget. It is runtime-source
adjacent because it exercises the existing demand-driven source collection with
more than one metric selection in a single Stream Deck action.

Read this folder before changing multi-slot settings, multi-metric action
subscriptions, dense progress-list rendering, or Property Inspector metric-slot
editing.
